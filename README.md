# Project Description
This project is a Python-based automation script that organizes files into different folders based on their types such as Images, Documents, Videos, and Audio. It helps in managing files efficiently and reduces manual effort.

# Features
- Automatic file organization  
- Categorization based on file types  
- Support for Images, Documents, Videos, Audio  
- Logging of operations  
- Error handling  

# Technologies Used
- Python  
- OS module  
- Shutil module  
- Datetime module  

# How to Run
1. Open terminal  
2. Run: python main.py  
3. Enter folder path when asked   

# Conclusion
This project demonstrates how automation can simplify file management tasks using Python.
